Welcome to Macro-2 Page.

Macro-2 is the second version of a simple mouse and keyboard recording program. It records keyboard and mouse input and then plays it back in the exact same way that it was recorded.

Current Features

Record Mouse input (keyboard in the works) Records the position of the mouse at a frequency of 500hz, resulting in smooth playback. Save/Open Recorded Macros Macros that have been recorded can be saved to a text file, or previous ones loaded. Hotkeys Control the macro recording and playback with the use of hot-keys.
